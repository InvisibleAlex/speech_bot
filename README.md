Telegram Бот переводит напечатанный текст в речь. Используются 2-а голосовых движка: русский и английский 
(находятся в Win10).
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Speech\Voices\Tokens\TTS_MS_RU-RU_IRINA_11.0
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Speech\Voices\Tokens\TTS_MS_EN-US_DAVID_11.0

Есть простой сомодельный фильтр, который отличит русский текст сообщения от английского и автоматически
подставит нужный языковой движек. Также, если бот увидит язык, кторый не похож на русский или англайский, 
то он выдаст сообщение об ошибке, чтобы не искажать неподходящим движком текст сообщения.

В боте НЕ использовался кодировщик mp3 -> ogg в виду того, что c выходом новой версии эти вещи автоматически
стали делатся на серверах телеграма.

Запуск бота:


1. После скачивание проекта, добавьте бота в телеграм @Smart_blondy_bot
2. Нужно создать пустой файл TOKEN.txt в той же папке, где лежат файлы bot.py, voice_ru.py, voice_ru.py, voice_eng.py
2. Открыть TOKEN.txt и скопировать токен для доступа бота к телеграмму и сохранить
3. В этой же папке создать пустую папку с именем data. Там будет хнанится последнее русское или английское сообщение *.mp3
4. Запустить файл bot.py и бот будет работать
5. Если бот не запустился, выполнить команду pip install python-telegram-bot==20.0a0
